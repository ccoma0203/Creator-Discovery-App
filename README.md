# 📱 Creator Discovery — Finding Creators Who Deserve More Eyes

"Posting consistently, but nobody's watching." 
I wanted to change that.

## 💡 The Idea

A project to automatically discover small creators 
(under 100 followers) who post consistently —
and introduce them to the world through a dedicated SNS account.

## 🎯 Goals

- [ ] Build an algorithm to identify consistently uploading small creators
- [ ] Filter by upload frequency, follower count, and content quality
- [ ] Run an SNS account that introduces discovered creators
- [ ] Eventually expand into a standalone app

## 🛠️ Planned Stack

- Algorithm built with Claude Code
- YouTube / Instagram / TikTok API
